TimeSpaceExplorer

A.H
Informations :

Cette montre fait parti de la collection TimeSpaceExplorer. Le modèle A.H vous permet de voyager dans le temps, que ce soit dans le futur ou dans le passé!
Activation : Appuyez sur le bouton de mise en marche situé sur le côté de la montre pour l'activer.
Sélection de la Date et de l'Heure : Utilisez l'écran tactile pour sélectionner la date et l'heure à laquelle vous souhaitez voyager. La montre vous montrera un aperçu de cette époque.
Appuyez sur "Voyager" : Une fois la date et l'heure choisies, appuyez sur le bouton "Voyager" pour initier le voyage dans le temps.
Voyage Temporel : Vous serez instantanément transporté à l'époque sélectionnée. Profitez de votre voyage dans le temps, mais soyez conscient des conséquences de vos actions.
Retour dans le Présent : Pour revenir à votre époque actuelle, appuyez sur le bouton "Retour" et sélectionnez l'option "Retour au Présent" sur l'écran.
Conseils: Assurez-vous que la montre est chargée pour éviter d'être coincé dans une autre époque.

F.R.B
Informations :

Activation : Appuyez sur le bouton de mise en marche situé sur le côté de la montre pour l'activer.
Sélection de la Destination : Utilisez l'écran tactile pour sélectionner la destination vers laquelle vous souhaitez vous téléporter. Vous pouvez également entrer des coordonnées spécifiques.
Appuyez sur "Téléporter" : Une fois la destination choisie, appuyez sur le bouton "Téléporter" pour initier le processus de téléportation.
Téléportation : Vous serez instantanément téléporté à la destination choisie. Assurez-vous de rester immobile pendant le processus.
Retour à l'Emplacement d'Origine : Pour revenir à l'emplacement où vous étiez précédemment, appuyez sur le bouton "Retour" et sélectionnez l'option "Retour EO" sur l'écran.

PM
Informations :

Activation : Appuyez sur le bouton de mise en marche situé sur le côté de la montre pour l'activer.
Sélection du Mode : Utilisez l'écran tactile pour choisir entre les modes "Voyage Temporel" et "Téléportation".
Voyage Temporel : Si vous choisissez le mode "Voyage Temporel", sélectionnez la date, l'heure et la destination temporelle souhaitées, puis appuyez sur "Voyager" pour vous déplacer dans le temps et l'espace.
Téléportation : Si vous optez pour le mode "Téléportation", sélectionnez la destination spatiale et appuyez sur "Téléporter" pour vous déplacer instantanément où vous le souhaitez dans le monde.
Retour à l'Emplacement d'Origine : Pour revenir à votre emplacement d'origine ou à votre époque actuelle, utilisez le bouton "Retour".
